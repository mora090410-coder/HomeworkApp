# UI Component Rules & Patterns

## Buttons
*   **Primary:** Cardinal (#990000) BG, White Text. Hover: Darker Cardinal (#7A0000). Focus: Gold (#FFCC00) ring.
*   **Secondary:** White BG, Cardinal Text, Light Gray (#CCCCCC) Border. Hover: Light Neutral (#F5F5F5).

## Cards
*   **Background:** White.
*   **Border:** Light Gray (#CCCCCC) 1px.
*   **Typography:** Title in Black, Supporting text in Dark Gray.
*   **Highlight:** Thin Gold top border or Gold badge.

## Gradients
*   **Primary:** Cardinal → Gold (#990000 to #FFCC00). Use for Hero backgrounds, banners.
*   **Accent:** Tertiary pairings (e.g., #E43D30 → #FDE021). Use for illustrations only.

## Typography Rules
*   **Headings:** Serif (Libre Baskerville). Authority.
*   **Body:** Sans (Inter). Clarity.
*   **Emphasis:** Use Gold for badges/backgrounds, NOT text color.

## Disallowed Combinations
*   Gold text on White.
*   White text on Gold.
*   Cardinal text on Black.
*   Tertiary colors as body text.
